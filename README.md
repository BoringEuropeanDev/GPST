# рҹ§  Prediction Engine

GPST currently supports **two prediction architectures**.

---

# v1 вҖ” Rules-Based Model

A deterministic weighted scorecard.

### Signal Weights

| Category             | Weight |
| -------------------- | ------ |
| Technical Indicators | 40%    |
| News Sentiment       | 30%    |
| Macro Regime         | 20%    |
| Sector Strength      | 10%    |

### Technical Indicators Used

* RSI
* MACD
* SMA crossovers
* Bollinger Bands
* Momentum
* Volume trends

### Output

```
Weighted Score вҶ’ Threshold вҶ’ Prediction
```

Result:

```
UP
DOWN
NEUTRAL
```

Confidence increases when **multiple signals align**.

---

# v2 вҖ” Machine Learning Pipeline (WIP)

A supervised learning model designed to **augment or replace the rules system**.

### Current Status

* Feature dataset stored as **Parquet**
* Built using **scikit-learn**
* Historical signals used as features
* Labels derived from **actual price direction**

### Planned Improvements

* nightly retraining
* ensemble models
* feature importance analysis
* hyperparameter optimization
* rolling backtests

Relevant modules:

```
backend/app/services/prediction_engine.py
backend/app/services/ml_pipeline.py
```

# рҹҡҖ Local Development

## Prerequisites

* Python **3.11+**
* Node **18+**
* Git

---

# Backend Setup

```bash
cd backend

python -m venv venv

# Linux / macOS
source venv/bin/activate

# Windows
venv\Scripts\activate

pip install -r requirements.txt

cp .env.example .env
```

Run API:

```bash
uvicorn main:app --reload --port 8000
```

API docs:

```
http://localhost:8000/docs
```

---

# Frontend Setup

```bash
cd frontend

npm install
npm start
```

Open:

```
http://localhost:3000
```
